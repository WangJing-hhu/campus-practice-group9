## 验证方法

全部 50 个页面通过 Python `requests` 库逐一执行 HTTP GET 请求验证：

- 请求头：User-Agent 模拟 Chrome 浏览器
- 超时设置：15 秒
- 跟随重定向
- 验证标准：HTTP 200 + 域名属于 hhu.edu.cn 子域 + 去除 HTML 标签后正文 ≥ 200 字符
