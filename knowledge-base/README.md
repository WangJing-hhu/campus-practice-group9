# 河海大学高校知识库

本知识库从 `site-snapshot/` 静态快照中提取而成，适合用于检索、问答、归档。

## 目录

- `01_学校概况/`：学校简介、大学章程、历史名人、历任领导、现任领导、校园文化等
- `02_组织机构/`：党政职能部门、群团组织、直属单位、派出机构、院系设置等
- `03_教育教学/`：师资队伍、本科生/研究生/留学生培养、终身教育等
- `04_科学研究/`：科研动态、学术会议、科研机构、科研成果、学术期刊等
- `05_招生就业/`：本科生/研究生/留学生招生、就业指导等
- `06_人才招聘/`：招聘信息
- `07_信息公开/`：信息公告、信息公开相关栏目
- `08_新闻动态/`：历年新闻、校园动态、媒体河海等
- `99_其他/`：未自动归类的页面
- `99_附件/`：PDF、DOC、DOCX、ZIP 等原始附件
- `INDEX.md`：全部页面与附件索引

## 统计

- 提取网页：1097 篇
- 复制附件：65 个

## 使用建议

当前应用递归读取 Markdown 文件建立索引；`99_附件/` 保存难以重新收集的原始附件，虽然不直接参与索引，但作为资料来源完整保留。
