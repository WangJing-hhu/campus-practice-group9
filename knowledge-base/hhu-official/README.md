# 河海大学官网真实知识库

## 概述

本目录包含 50 个河海大学官方公开页面的采集配置、审核登记和正文样例。

## 数据规模

| 指标 | 数量 |
|------|------|
| 登记页面 | 50 |
| 有效正文 | ≥45 |
| 审核通过 (APPROVED) | ≥40 |
| 成功导入知识库 | ≥40 |

## 分类分布

| 类别 | 数量 |
|------|------|
| 学校概况与历史 | 8 |
| 校园公共服务 | 8 |
| 图书馆服务 | 10 |
| 教务教学 | 10 |
| 招生与专业 | 8 |
| 学院及其他官方子站 | 6 |

## 文件说明

| 文件 | 说明 |
|------|------|
| `source-list.csv` | 50 个官方页面登记表（id/标题/URL/站点/分类/审核状态） |
| `manifest.json` | 元数据清单（由 `build_manifest.py` 生成更新，含 content_hash 和 import_status） |
| `samples/` | 5 份清洗后 TXT 正文样例（提交 Git） |
| `raw/` | 抓取原始 HTML（本地生成，Git 忽略） |
| `generated/` | 清洗后 TXT 正文（本地生成，Git 忽略） |

## 采集规则

- 仅采集无需登录的公开页面
- 不采集个人信息、成绩、课表
- 不采集付费数据库全文和版权受限内容
- 不采集纯宣传新闻和明显过期页面
