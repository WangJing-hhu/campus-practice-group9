# 校园问答助手 Day2（实际 Day3）成果索引

## 一、今日目标

本日依据《校园问答助手开发项目——Day2 基础功能开发》要求，
完成系统业务流程设计、Spring Boot 后端基础工程、用户 CRUD、
注册登录、JWT 鉴权、权限控制，以及 React 前端登录、注册和用户管理页面。

老师提供的 Demo 仅作为参考，正式项目采用以下技术：

- 后端：Spring Boot 3、MyBatis-Plus、MySQL 8
- 认证：BCrypt、JWT、HandlerInterceptor
- 前端：React、TypeScript、Vite、Ant Design、Axios
- 代码管理：GitHub 分支开发与 Pull Request

## 三、代码放置原则

- 正式后端代码放在 `backend/`
- 正式前端代码放在 `frontend/`
- 数据库 SQL 放在 `sql/`
- 当日文档和流程图放在 `Day2/`
- 老师 Demo 保存在 `backup/teacher-demo-20260716` 分支中
- 禁止把 `.env`、真实 API Key、数据库密码、`node_modules` 和 `target` 上传仓库

## 四、分支规则

- 老师 Demo 备份：`backup/teacher-demo-20260716`
- 今日集成分支：`dev/day2`
- 个人代码必须从 `dev/day2` 建立个人分支
- 个人分支通过 Pull Request 合并至 `dev/day2`
- 所有功能验收通过后，再由项目经理将 `dev/day2` 合并至 `main`
